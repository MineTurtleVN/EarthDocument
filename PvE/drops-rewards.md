# 🎁 Drop Và Phần Thưởng PvE

Plugin chính: `ExcellentCrates`, `TurtleDungeon`, `MMOItems`

📁 Config: /plugins/ExcellentCrates/crates/dungeons.yml
📁 Config: /plugins/ExcellentCrates/keys/dungeon.yml
📁 Config: /plugins/MMOItems/drops.yml

Nguồn thưởng PvE quan trọng nhất là dungeon, key dungeon và crate `dungeons`.

| Nguồn | Config | Ghi chú |
|---|---|---|
| Dungeon key | `/plugins/ExcellentCrates/keys/dungeon.yml` | Key mở crate dungeon |
| Dungeon crate | `/plugins/ExcellentCrates/crates/dungeons.yml` | Phần thưởng dungeon chính |
| MMOItems drops | `/plugins/MMOItems/drops.yml` | Drop item theo MMOItems |
| Trang bị ngẫu nhiên | `/plugins/ExcellentCrates/crates/trangbingaunhien.yml` | Crate trang bị lớn, file rất lớn |

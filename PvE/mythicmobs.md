# 👹 MythicMobs Và Mob Dungeon

Plugin chính: `TurtleDungeon`, `MythicMobs`

📁 Config: /plugins/TurtleDungeon/config.yml → mob-display-names
📁 Config: /plugins/MythicMobs/

`TurtleDungeon` map internal mob ID sang tên tiếng Việt để hiển thị cho người chơi.

| Internal ID | Tên hiển thị |
|---|---|
| MagmaWisp | Hồn Đá Vất Vưởng |
| StoneSentinel | Vệ Binh Hóa Thạch |
| AncientAltarGuardian | Thần Oán Hận |
| EarthfireSymphony | Giao Hưởng Của Đất Và Lửa |
| BloodRiteHighPriest | Tư Tế Huyết Lễ |
| AbyssHerald | Sứ Giả Vực Sâu |
| ThronelessKing | Vua Không Ngai |
| BloodRitualOverlord | Tế Ma Vương |
| EdenSacredSpirit | Linh Vườn Địa Đàng |

Người chơi nên đọc tên mob theo GUI/dungeon thay vì internal ID, nhưng admin cần giữ ID khi chỉnh drop/spawn/skill.

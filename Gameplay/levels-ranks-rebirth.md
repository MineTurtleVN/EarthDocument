# 🧬 Level, Rank Và Mốc Sức Mạnh

Plugin chính: `TurtleLevel`, `zMenu`

📁 Config: /plugins/TurtleLevel/config.yml
📁 Config: /plugins/TurtleLevel/gui.yml
📁 Config: /plugins/zMenu/inventories/RANK.yml

`TurtleLevel` lưu cấu hình level và GUI riêng. `zMenu` có `RANK.yml`, cho thấy server có màn hình rank/mốc sức mạnh để người chơi tra cứu.

| Thành phần | Ý nghĩa |
|---|---|
| TurtleLevel `config.yml` | Công thức và mốc level lớn, file rất dài nên nên xem theo từng key khi chỉnh |
| TurtleLevel `gui.yml` | GUI xem tiến độ level |
| zMenu `RANK.yml` | Menu rank/player-facing rank |

Gợi ý chơi: luôn kiểm tra GUI level/rank trước khi tiêu nguyên liệu lớn, vì nhiều mốc dungeon/trang bị thường phụ thuộc vào sức mạnh hiện tại.

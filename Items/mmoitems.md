# ⚔️ MMOItems

Plugin chính: `MMOItems`

📁 Config: /plugins/MMOItems/item-types.yml
📁 Config: /plugins/MMOItems/item-tiers.yml

Loại item Việt hóa đáng chú ý:

| Type ID | Tên | Vai trò |
|---|---|---|
| `MU` | Mũ | Giáp đầu |
| `AO` | Áo | Giáp ngực |
| `QUAN` | Quần | Giáp chân |
| `GIAY` | Giày | Giáp chân |
| `SWORDNEW` | Kiem | Vũ khí kiếm mới |
| `BOW_NEW` | Bow New | Cung mới |
| `RIU` | Rìu | Vũ khí rìu |
| `CUP` | Cúp | Công cụ đào |
| `CUOCSANHO` | Cuốc San Hô | Công cụ đặc thù |
| `MATERIAL` | Material | Vật liệu |
| `NGOCTHIENSU` | Ngọc Thiên Sứ | Consumable đặc biệt |

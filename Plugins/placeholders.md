# 🧩 Placeholders

Plugin chính: `PlaceholderAPI`, `ExcellentCrates`, `zMenu`

📁 Config: /plugins/PlaceholderAPI/
📁 Config: /plugins/zMenu/global-placeholders.yml
📁 Config: /plugins/ExcellentCrates/config.yml → Holograms.TemplateList

ExcellentCrates dùng placeholder dạng:

| Placeholder | Ý nghĩa |
|---|---|
| `%crate_name%` | Tên crate |
| `%excellentcrates_keys_%crate_id%%` | Số key người chơi có cho crate |
| `%excellentcrates_next_milestone_openings_%crate_id%%` | Số lần mở còn tới mốc tiếp theo |

zMenu có `global-placeholders.yml`, nên các menu có thể dùng placeholder chung để thống nhất hiển thị.

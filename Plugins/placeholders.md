## 👤 Player/Menu Placeholders

| Placeholder | Ý nghĩa |
|---|---|
| `%player_name%` | Tên người chơi trong menu chính |
| `%player_first_played_formatted%` | Ngày vào server lần đầu |
| `%vault_eco_balance_formatted%` | Tiền Vault định dạng đẹp |
| `%vault_eco_balance_fixed%` | Tiền Vault dạng số, dùng requirement repair |
| `%playerpoints_points%` | Xu/PlayerPoints, dùng Xu Shop và menu chính |
| `%player_max_health_rounded%` | Máu tối đa làm tròn |
| `%statistic_player_kills%` | Số kill statistic |
| `%statistic_deaths%` | Số lần chết |
| `%luckperms_has_permission_essentials.repair%` | Check quyền repair hand |
| `%luckperms_has_permission_essentials.repair.all%` | Check quyền repair all |

## 🏪 ExcellentShop Internal Placeholders

| Placeholder | Ý nghĩa |
|---|---|
| `%balance%` | Số tiền trong shop main menu |
| `%sell_multiplier%` | Hệ số bán theo rank/permission |
| `%shop_icon_name%` | Tên icon shop |
| `%shop_description%` | Mô tả shop |
| `%product_price_buy_formatted%` | Giá mua định dạng |
| `%product_price_sell_formatted%` | Giá bán định dạng |
| `%product_price_avg_diff_buy%` | Biến động giá mua trung bình |
| `%product_price_avg_diff_sell%` | Biến động giá bán trung bình |
| `%product_stock_global_buy_amount_left%` | Stock mua còn lại toàn server |
| `%product_stock_player_buy_amount_left%` | Giới hạn mua còn lại của người chơi |

# ⌨️ Commands Và Permissions

Plugin chính: `ExcellentShop`, `zMenu`, `PvPManager`, `LuckPerms`

📁 Config: /plugins/ExcellentShop/config.yml
📁 Config: /plugins/zMenu/inventories/
📁 Config: /plugins/PvPManager/config.yml

| Lệnh/Menu | Nguồn | Ghi chú |
|---|---|---|
| `/vshop` | ExcellentShop VirtualShop | Mở shop ảo |
| zMenu menu chính | `MENU.yml` | Có thể được bind qua command zMenu |
| zMenu warp | `WARPS.yml` | Mở danh sách warp |
| zMenu dungeon | `DUNGEONS.yml` | Mở chọn dungeon |
| PvP toggle | PvPManager | Có cooldown 15 giây nếu dùng toggle |
| LuckPerms permission | LuckPerms | Rank/quyền thực tế nằm trong LuckPerms storage |

Ghi chú: tài liệu này không liệt kê permission node cụ thể nếu chưa đọc storage LuckPerms để tránh đoán sai.

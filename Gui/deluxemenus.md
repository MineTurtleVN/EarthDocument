# 📋 zMenu Inventories

Plugin chính: `zMenu`

📁 Config: /plugins/zMenu/inventories/

Các GUI quan trọng đã thấy trong thư mục `inventories`:

| File | Mục đích dự đoán |
|---|---|
| `MENU.yml` | Menu chính |
| `HUONGDAN.yml` | Hướng dẫn |
| `WARPS.yml` | Warp |
| `RTP.yml` | Random teleport |
| `DUNGEONS.yml` | Menu dungeon tổng |
| `DUNGEON_D1.yml` đến `DUNGEON_D5.yml` | Menu dungeon từng cấp |
| `KIT.yml` | Kit |
| `RANK.yml` | Rank |
| `EVENT.yml`, `EVENT_BOOSTER.yml`, `EVENT_MOCNAP.yml` | Event/booster/mốc nạp |
| `MOCHOM.yml` | Mở hòm/crate |
| `CSHOP.yml` | Shop tùy chỉnh |
| `DONATE/` | Donate |

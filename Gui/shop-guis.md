# 🛒 GUI Shop Và Crate

Plugin chính: `ExcellentShop`, `ExcellentCrates`, `zMenu`

📁 Config: /plugins/ExcellentShop/virtual_shop/main.menu.yml
📁 Config: /plugins/ExcellentShop/virtual_shop/sell.menu.yml
📁 Config: /plugins/ExcellentCrates/ui/
📁 Config: /plugins/zMenu/inventories/MOCHOM.yml

| GUI | Cách dùng |
|---|---|
| ExcellentShop main menu | Mở `/vshop` để mua/bán |
| ExcellentShop sell menu | Bán vật phẩm theo shop |
| ExcellentCrates preview/opening | Xem phần thưởng, mở hòm |
| zMenu `MOCHOM.yml` | Menu mở hòm theo server |

ExcellentCrates click action: chuột trái xem preview, chuột phải mở crate, shift phải mass open.
